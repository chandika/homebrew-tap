# Homebrew Tap

Homebrew formulae for tools by [@chandika](https://github.com/chandika).

## Install

```bash
brew tap chandika/tap
brew install mirage-proxy
```

## Packages

| Formula | Description |
|---------|-------------|
| [mirage-proxy](https://github.com/chandika/mirage-proxy) | Invisible sensitive data filter for LLM APIs |
